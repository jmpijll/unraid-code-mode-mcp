# CLAUDE.md

This file exists so **Claude Code** picks up the same per-repo
guidance as every other agent platform that reads `AGENTS.md`.

> **Read [`AGENTS.md`](AGENTS.md) instead — it is the canonical
> contributor guide.** Everything below is a one-line redirect; no
> guidance is duplicated here.

The choice to keep the canonical file as `AGENTS.md` (rather than
`CLAUDE.md`) is deliberate: `AGENTS.md` is the cross-platform
convention adopted by Cursor, opencode, Codex, and others; Claude Code
also reads `AGENTS.md` natively as of recent versions, but the
top-level `CLAUDE.md` is its preferred name and we want Claude Code
sessions to find guidance on the very first directory listing.

If you're adopting the **runtime persona** (the "Unraid expert agent"
that drives the MCP server), see
[`examples/unraid-expert-agent/AGENTS.md`](examples/unraid-expert-agent/AGENTS.md)
— that's the system-prompt-shaped file you load into your agent, not
this one.
